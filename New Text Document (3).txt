
Technical Proposal (العرض الفني)
1. نبذة عن المشروع:
سيتم تطوير نظام إدارة مدرسة باستخدام إطار العمل ASP.NET. يهدف النظام إلى تسهيل إدارة البيانات الأكاديمية والإدارية للمدرسة، بما في ذلك إدارة الطلاب، الموظفين، الدرجات، الجداول، والحضور.

2. المميزات الرئيسية:
إدارة الطلاب: إضافة، تعديل، وحذف بيانات الطلاب.
إدارة الموظفين: تسجيل وتحديث بيانات المعلمين والموظفين الإداريين.
إدارة الدرجات: تسجيل الدرجات وإصدار التقارير الأكاديمية.
إدارة الحضور: تسجيل حضور الطلاب والموظفين.
الجدول الزمني: تنظيم الجداول الدراسية.
تقارير: إنشاء تقارير تحليلية وإحصائية.
3. التقنيات المستخدمة:
ASP.NET Core: كإطار العمل الرئيسي لتطوير النظام.
SQL Server: لإدارة قواعد البيانات.
HTML, CSS, JavaScript: لتطوير الواجهة الأمامية.
Bootstrap: لتحسين التصميم وجعله متجاوباً مع جميع الأجهزة.
Entity Framework: للتعامل مع قاعدة البيانات.
4. الجدول الزمني:
مرحلة التحليل وتصميم النظام: 1 أسبوع.
تطوير الواجهة الأمامية: 2 أسابيع.
تطوير الجزء الخلفي (Back-end): 3 أسابيع.
اختبار النظام وضمان الجودة: 1 أسبوع.
التسليم النهائي: 7 أسابيع إجمالاً.
